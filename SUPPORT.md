# Support

This document describes how to get help with nx-astro.

## Where to Ask Questions

| Channel                  | Best For                                      | Link                                                              |
| ------------------------ | --------------------------------------------- | ----------------------------------------------------------------- |
| **GitHub Discussions**   | General questions, how-to help, brainstorming | [Discussions](https://github.com/geekvetica/nx-astro/discussions) |
| **GitHub Issues**        | Bug reports, feature requests                 | [Issues](https://github.com/geekvetica/nx-astro/issues)           |
| **Nx Community Discord** | Nx-specific questions and ecosystem help      | [Discord](https://go.nx.dev/community)                            |
| **Astro Discord**        | Astro-specific questions                      | [Discord](https://astro.build/chat)                               |

## Issues vs Discussions

**Use Discussions when:**

- You have a question about how to use nx-astro
- You want to brainstorm an idea before filing a feature request
- You want to share something you built with nx-astro
- You're not sure if something is a bug

**Use Issues when:**

- You've found a reproducible bug
- You have a well-defined feature request
- You want to track specific work

## Response Expectations

- **Issues**: We aim to triage new issues within a few days. Complex issues may take longer.
- **Discussions**: Community members and maintainers respond as available. No guaranteed response time.
- **Pull Requests**: We aim to review PRs within a week.
